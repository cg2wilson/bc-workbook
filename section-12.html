    Section 1.2 Function Behavior and End Behavior Limits

    Subsection 1.2.1 Descriptions of Function Behavior

    Especially as we consider longer-term models, we are concerned about what happens to a function as time advances.

    Definition 1.2.1. Increasing/Decreasing/Constant.

    Let \(f\) be a function defined over some input interval. The function is said to be
    • increasing if the output values increase on the interval
    • decreasing if the output values decrease on the interval
    • constant if the output values remain the same on the interval

    Example 1.2.2.

    On what intervals is \(g(x) = -x^3 + 16x -5\) increasing, decreasing, or constant? Use the calculator to help you find the answer.
    Solution.
    The function is increasing on \((-2.42,2.72)\) and decreasing on \((-\infty,-2.42)\cup (2.72,\infty)\)

    Example 1.2.3.

    Is the function given in the table below increasing, decreasing, or constant? Why?
    \(x\) 2 4 6 8 10
    \(h(x)\) 5 6 8 12 20
    Solution.
    The function is increasing; as the input increases, so does the output.

    Example 1.2.4.

    The function \(f(x) = c\) is constant. Look at the graph and/or table and explain why.
    Solution.
    Answers and explanations may vary

    Subsection 1.2.2 Direction and Curvature

    Definition 1.2.5. Concavity.

    A function \(f\) defined over an input interval is said to be
    • concave up if a graph of the function appears to be bending upward
    • concave down if a graph of the function appears to be bending downward
    The curvature of a function is called concavity.

    Example 1.2.6.

    Describe the concavity of \(k(t)=-t^2 + 8t -13\text{.}\) Does it appear to ever change? If so, where? Draw a picture.
    Solution.
    \(k\) is concave down; how you choose to sketch the picture is up to you, whether it's done by hand or calculator.

    Example 1.2.7.

    Describe the concavity of \(f(z) = \ln z\text{.}\) Does it appear to ever change? If so, where? Draw a picture.
    Solution.
    \(f\) is concave down; how you choose to sketch the picture is up to you, whether it's done by hand or calculator.

    Example 1.2.8.

    Describe the concavity of \(g(x) = -2x\text{.}\) Does it appear to ever change? If so, where? Draw a picture.
    Solution.
    \(g\) is neither concave down nor concave up; how you choose to sketch the picture is up to you, whether it's done by hand or calculator.

    Example 1.2.9.

    Describe the concavity of \(h(d) = d^3 - 11d^2 + 38d - 37\text{.}\) Does it appear to ever change? If so, where? Draw a picture.
    Solution.
    \(h\) is concave down on about \((-\infty,3.79)\) and concave up on about \((3.79,\infty)\text{;}\) how you choose to sketch the picture is up to you, whether it's done by hand or calculator.

    Definition 1.2.10. Inflection Point.

    A point on a continuous function where the concavity of the function changes is called an inflection point.

    Example 1.2.11.

    The function \(P(t)=2t^3-10t^2-3t + 275\) describes the profit (in hundred dollars) made by a small business after a rash of bad Yelp reviews where \(t\) is the number of weeks since the reviews were put online. Let \(0 \leq t \leq 5\text{.}\)
    1. Sketch a picture of the graph.
    2. Estimate the input and output values at the inflection point(s).
    3. Identify the intervals where \(P\) is increasing, decreasing, and constant.
    4. Identify the intervals where \(P\) is concave up, concave down, or neither.
    5. Use the information from (a)-(c) to describe what was happening to the profit made by the business between the first and sixth week.
    Solution.
    2. The inflection point is at about \(t=1.97\) weeks, and \(P(1.97) = 245.58\) hundred dollars.
    3. The function appears to be decreasing on \([0,3.48)\) and increasing on \((3.48,5]\)
    4. The function appears to be concave down on \([0,1.97)\) and concave up on \((1.97,5]\)
    5. Starting at week 1, the profit is decreasing until approximately \(t=3.48\text{,}\) at which point it begins increasing and stays increasing through week 6.
      This is probably supposed to be either "week 1 to week 5" or "week 0 to week 5", but that will be fixed in future iterations.

    Subsection 1.2.3 Limits and End Behavior

    Definition 1.2.12. End Behavior.

    The end behavior of a function refers to the behavior of the output values of the function as the input values become larger and larger, or smaller and smaller.
    As the input values become larger and larger (more and more positive), we say that the input increases without bound. As they become smaller and smaller (more and more negative), the input decreases without bound.

    Example 1.2.13.

    Consider \(h(d) = d^3 - 11d^2 + 38d - 37\text{.}\)
    1. Sketch the function on the interval \([0,6]\text{.}\)
    2. In a sentence, describe the end behavior of \(h\) as the input increases without bound.
    3. In a sentence, describe the end behavior of \(h\) as the input decreases without bound.
    Solution.
    2. As the input increases without bound, the output increases without bound.
    3. As the input decreases without bound, the output decreases without bound.
    There are three possibilities when we consider the end behavior of a function:
    • The output values may approach or equal a certain number
    • The output values may increase or decrease without bound
    • The output values may oscillate and fail to approach any particular number

    Example 1.2.14.

    Draw three functions that have will have each of these three end behaviors.
    Solution.
    This is up to you; it depends on what you want to emphasize.

    Example 1.2.15.

    Determine the end behavior/limit of the function \(f(x) = \dfrac{2x}{x-1}\) as the input increases without bound, using numerical estimation. Record your approximations with full decimal accuracy, and round the final answer to the hundredths.
    \(x\) \(f(x) = \dfrac{2x}{x-1}\)
    \(10\)
    \(100\)
    \(1000\)
    \(10000\)
    \(100000\)
    End Behavior/Limit
    Solution.
    \(x\) \(f(x) = \dfrac{2x}{x-1}\)
    \(10\) 2.2222222222222
    \(100\) 2.020202020202
    \(1000\) 2.002002002002
    \(10000\) 2.000200020002
    \(100000\) 2.0000200002
    End Behavior/Limit 2.00

    Note.

    When creating a table, you need to stop when the digit after the one you're rounding to repeats twice.

    Definition 1.2.16. Limit.

    A function \(f\) is said to have a limit \(L\) if the output of \(f\) approaches \(L\) as the input approaches some (possibly infinite) value \(a\text{.}\) We write this using the following notation:
    \begin{equation*} \lim_{x\to a} f(x) = L \end{equation*}

    Example 1.2.17.

    Rewrite the end behavior/limit from the previous example using limit notation.
    Solution.
    \(\ds \lim_{x\to \infty} \dfrac{2x}{x-1} = 2\)

    Definition 1.2.18. Horizontal Asymptote.

    A horizontal line with the equation \(y = L\) is called a horizontal asymptote.

    Example 1.2.19.

    Let \(f(x) = x^2\) and \(g(x) = x^3\text{.}\)
    1. Write the statement "The limit of \(f(x)\) as \(x\) approachs \(\infty\) is \(\infty\)" in limit notation.
    2. Find \(\ds \lim_{x\to -\infty} f(x)\text{,}\) and write the notation in words (like in (a))
    3. Find the end behaviors of \(g(x)\text{,}\) and write them in limit notation.
    Solution.
    1. \(\displaystyle \ds \lim_{x\to \infty} f(x) = \infty\)
    2. \(\ds \lim_{x\to -\infty} f(x) = \infty\text{;}\) the limit of \(f(x)\) as \(x\) decreases without bound is infinity.
    3. \(\ds \lim_{x\to -\infty}g(x) = -\infty\) and \(\ds \lim_{x\to \infty} g(x) = \infty\)

    Example 1.2.20.

    Sketch the following functions, and use the sketches to find the limit as the input increases without bound and decreases without bound:
    1. \(\displaystyle f(x) = \ln x\)
    2. \(\displaystyle g(x) = e^x\)
    3. \(\displaystyle h(x) = \dfrac{1}{1 + e^x}\)
    Solution.
    1. \(\ds \lim_{x\to \infty} f(x) = \infty\text{.}\)
      There's two ways you might approach the vertical asymptote at 0 here; one way is to say that it's impossible to determine what happens as the inptu decreases without bound. The other way is to note that we can only do something like take the limit as \(x\to 0^+\text{;}\) this sets up for Section 1.3, but might happen a little early for everyone to make the connection.
    2. \(\ds \lim_{x\to \infty} g(x) = \infty\) and \(\ds \lim_{x\to -\infty} g(x) = 0\)
    3. \(\ds \lim_{x\to \infty} h(x) = 0\) and \(\ds \lim_{x\to -\infty} g(x) = 1\)

    Example 1.2.21.

    Use numerical estimation to find \(\ds\lim_{x\to \infty} (1-0.6^x)\text{.}\) Make a table showing at least five inputs and the corresponding outputs; write all decimals in the table, and round your final answer to two decimal places. Start your input at 2 and double.
    Solution.
    \(x\) \(1-0.6^x\)
    2 0.64
    4 0.8704
    8 0.98320384
    16 0.99971788900926
    32 0.99999992041339
    This one converges fast, and it might be helpful to show them a graph of the exponential. If they remember the properties of exponentials from 1643, then this should be a relatively straightforward question.
    A question students may have is why we're stopping at 32 if the input is supposed to increase without bound. I can think of two responses: 1) our rounding rules tell us to stop when the third decimal repeats twice, and this condition is met after 32; 2) students are welcome to try more numbers to play around with the behavior, but should know that if this were a test question, they would need to stop after 32.

    Example 1.2.22.

    Use numerical estimation to find \(\ds \lim_{t\to -\infty} \lrpar{1+t^{-2}}\text{.}\) Make a table showing at least five inputs, and the corresponding outputs; write all decimals in the table, and round your final answer to the nearest integer. Start your input at \(-10\) and double.
    Solution.
    \(t\) \(1+t^{-2}\)
    \(-10\) 1.01
    \(-20\) 1.0025
    \(-40\) 1.000625
    \(-80\) 1.00015625
    \(-160\) 1.0000390625
    \(-320\) 1.000009765625
    In reality, we only need two lines here. I like to go out this far to emphasize the numerical behavior. Again, you'll want to emphasize that if this is a question on the test, they need to stop after the second line (since the first decimal place repeats after two iterations).
