Definition 2.2.7. Percentage Rate of Change.

For smooth, continuous function \(f\text{,}\) if the rate of change \(f'(x_0)\) exists for input value \(x_0\) and \(f(x_0)\neq 0\text{,}\) then we define the percentage rate of change as \(\dfrac{f'(x_0)}{f(x_0)}\cdot 100\text{.}\)
The output for percentage rate of change is "% per input"
