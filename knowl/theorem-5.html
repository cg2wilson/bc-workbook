Theorem 4.4.3. Second Derivative Test.

Let \(f(x)\) be a twice-differentiable function defined on input interval \([a,b]\) with \(a\lt c\lt b\)
  • If \(f'(c) = 0\) and \(f''(c)\gt 0\text{,}\) then \(f\) has a relative minimum at \(c\text{.}\)
  • If \(f'(c) = 0\) and \(f''(c)\lt 0\text{,}\) then \(f\) has a relative maximum at \(c\text{.}\)
