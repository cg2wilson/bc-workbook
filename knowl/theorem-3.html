Theorem 4.2.6. Old Derivative Facts.

Let \(f(x)\) be a function defined on an input interval \([a,b]\) and let \(a\lt c\lt b\)
  • If \(f'(x) \gt 0\text{,}\) then \(f(x)\) is increasing at \(x=c\text{.}\)
  • If \(f'(x) \lt 0\text{,}\) then \(f(x)\) is decreasing at \(x=c\text{.}\)
  • If \(f'(x) = 0\text{,}\) then the line tangent to \(f(x)\) at \(x=c\) is horizontal.
