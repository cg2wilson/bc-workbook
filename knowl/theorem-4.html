Theorem 4.4.2. Properties of the Second Derivative.

Let \(f(x)\) be a twice-differentiable function defined on input interval \([a,b]\) with \(a \lt c \lt b\)
  • \(f\) has an inflection point at \(x=c\) if and only if \(f''(c) = 0\) and changes sign
  • If \(f''(c)\) is positive, then \(f\) is concave up
  • If \(f''(c)\) is positive, then \(f'\) is increasing
  • If \(f''(c)\) is negative, then \(f\) is concave down
  • If \(f''(c)\) is negative, tthen \(f'\) is decreasing
