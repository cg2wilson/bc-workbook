Definition 4.2.1. Relative Extrema.

Let \(f(x)\) be a function defined on an input interval \([a,b]\text{.}\) Let \(a\lt c\lt b\text{.}\)
We say that \(f\) has a relative maximum at \(c\) if the output \(f(c)\) is bigger than any other output in some interval around \(c\text{.}\) Likewise, \(f\) has a relative minimum at \(c\) if the output \(f(c)\) is smaller than any other output in some interval around \(c\text{.}\)
