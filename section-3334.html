    Section 3.4 Rate of Change of Composite Functions

    Subsection 3.4.1 Review: Composite Functions

    Two functions \(f(x)\) and \(x(t)\) can be composed if and only if the output of \(x(t)\) is the input of \(f(x)\text{.}\) Notice how the notation is suggestive; \(f\) inputs \(x\text{,}\) which is exactly what \(x(t)\) outputs. We write the composition either as \((f\circ x)(t)\) or \(f(x(t))\text{.}\) The new input is now the input of \(x\) (ie, \(t\)), and the new output is the output of \(f\) (namely, \(f\)).

    Example 3.4.1.

    Identify the functions which make up the composite functions given below.
    1. \(\displaystyle f(x) = \dfrac{1}{x+2}\)
    2. \(\displaystyle g(x) = \ln (x^2)\)
    3. \(\displaystyle h(t) = e^{5t}\)
    4. \(\displaystyle q(x) = (2x+1)^5\)
    5. \(\displaystyle n(f) = \lrpar{3+\dfrac{1}{f}}^3\)
    6. \(\displaystyle s(h) = \ln \left(5h^2 + \dfrac{1}{h}\right)\)
    7. \(\displaystyle y(r) = \dfrac{5.317}{(2r^5 + 1.7)^2}\)
    8. \(\displaystyle w(c) = \sqrt[3]{\dfrac{c}{1+c}}\)
    9. \(\displaystyle f(x) = 1-\sqrt{e^x+5x}\)
    Solution.
    1. If \(f(x) = a(b(x))\text{,}\) we'll pick \(a(x) = \dfrac{1}{x}\) and \(b(x) = x+2\)
    2. If \(g(x) = a(b(x))\text{,}\) we'll pick \(a(x) = \ln (x)\) and \(b(x) = x^2\)
    3. If \(h(t) = a(b(t))\text{,}\) we'll pick \(a(t) = e^t\) and \(b(t) = 5t\)
    4. If \(q(x) = a(b(x))\text{,}\) we'll pick \(a(x) = x^5\) and \(b(x) = 2x+1\)
    5. If \(n(f) = a(b(g))\text{,}\) we'll pick \(a(f) = f^3\) and \(b(g) = 3+\dfrac{1}{f}\)
    6. If \(s(h) = a(b(h))\text{,}\) we'll pick \(a(h) = \ln h\) and \(b(h) = 5h^2 + \dfrac{1}{h}\)
    7. If \(y(r) = a(b(r))\text{,}\) we'll pick \(a(r) = \dfrac{5.317}{r^2}\) and \(b(r) = 2r^5 + 1.7\)
    8. If \(w(c) = a(b(c))\text{,}\) we'll pick \(a(c) = \sqrt[3]{c}\) and \(b(c) = \dfrac{c}{1+c}\)
    9. If \(f(x) = a(b(x))\text{,}\) we'll pick \(a(x) = 1-\sqrt{x}\) and \(b(x) = e^x+5x\)

    Subsection 3.4.2 The Chain Rule

    The chain rule is a rule for finding the derivative of composite functions. Let \(h(x) = f(g(x))\text{,}\) where the output of \(g\) is the input of \(f\text{.}\) Then,
    \begin{equation*} h'(x) = f'(g(x))\cdot g'(x) \end{equation*}
    The best way to learn the chain rule is with practice inside and outside of class.

    Subsection 3.4.3 Examples

    Example 3.4.2.

    For \(f(t) = 3t^2\) and \(t(x) = 4+7\ln x\text{,}\) find the rate of change function \((f\circ t)'(x)\) with respect to \(x\text{.}\)
    Solution.
    \((f\circ t)'(x) = 6(4+7\ln x)\cdot \dfrac{7}{x}\)

    Example 3.4.3.

    Let \(c(x) = 3x^2 - 2\) and \(x(t) = 4-6t\text{.}\) Find \(c'(t)\)
    Solution.
    \(c'(tO = 6(4-6t)\cdot (-6)\)

    Example 3.4.4.

    Consider the following functions:
    \begin{equation*} f(g) = \ln g \qquad g(h) = 5h + 2 \qquad h(j) = e^j\qquad j(x) = 4x^{-1} \end{equation*}
    Find \(f(x)\) and \(f'(x)\text{.}\)
    Solution.
    \(f(x) = \ln\lrpar{5e^{4x^{-1}}+2}\) and \(f'(x) = \dfrac{5e^{4x^{-1}}\cdot -4x^{-2}}{5e^{4x^{-1}}+2}\)

    Example 3.4.5.

    Find the derivative of \(f(x) = \dfrac{1}{x+2}\)
    Solution.
    \(f'(x) = -(x+2)^{-2}\)

    Example 3.4.6.

    Find the derivative of \(f(x) = \ln(x^2)\)
    Solution.
    \(f'(x) = \dfrac{1}{x^2}\cdot 2x = \dfrac{2}{x}\)

    Example 3.4.7.

    Find the derivative of \(f(x) = (\ln x) ^3\)
    Solution.
    \(f'(x) = 3(\ln x)^2\cdot \dfrac{1}{x}\)

    Example 3.4.8.

    Find the derivative of \(f(x) = e^{5x}\)
    Solution.
    \(f'(x) = 5e^{5x}\)

    Example 3.4.9.

    Find the derivative of \(f(x) = (e^x)^4\)
    Solution.
    \(f'(x) = 4e^{4x}\)

    Example 3.4.10.

    Find the derivative of \(f(x) = 7 + 5\ln (4x^2+3)\)
    Solution.
    \(f'(x) = \dfrac{5}{4x^2+3}\cdot 8x\)

    Example 3.4.11.

    If \(s(t) = 3e^{5t}\text{,}\) find \(s'(t)\)
    Solution.
    \(s'(t) = 15e^{5t}\)

    Example 3.4.12.

    Find the derivative of \(k(x) = 3e^{4x^2}\)
    Solution.
    \(k'(x) = 3e^{4x^2}\cdot 8x\)

    Example 3.4.13.

    Find the derivative of \(p(t) = (5+6e^{2t})^3\)
    Solution.
    \(p'(t) = 3(5+6e^{2t})^2\cdot 12e^{2t}\)

    Example 3.4.14.

    \(f(x) = 6(4x^2+3)^5\)
    Solution.
    \(f'(x) = 30(4x^2+3)^4\cdot 8x\)

    Example 3.4.15.

    \(f(x) = -12\ln (6x^2+3^x)\)
    Solution.
    \(f'(x) = \dfrac{-12}{6x^2+3^x}\cdot (12x + \ln 3\cdot 3^x)\)

    Example 3.4.16.

    \(f(x) = 2e^{0.5x} - 2x\)
    Solution.
    \(f'(x) = e^{0.5x} - 2\)

    Example 3.4.17.

    \(f(x) = \dfrac{7.2}{(4x^3+1)^4}\)
    Solution.
    \(f'(x) = 7.2\cdot 4(4x^3+1)^3\cdot 12x^2\)

    Example 3.4.18.

    \(f(x) = 3\sqrt{x^3 + 2\ln x}\)
    Solution.
    \(f'(x) = \dfrac{3}{2}\lrpar{x^3+2\ln x}^{-1/2}\cdot \lrpar{3x^2 + \dfrac{2}{x}}\)

    Example 3.4.19.

    Find the derivative of \(f(x) = e^{kx}\)
    Solution.
    \(f'(x) = ke^{kx}\)

    Example 3.4.20.

    Compute the derivative of \(e^{f(x)}\)
    Solution.
    \(f'(x)e^{f(x)}\)

    Example 3.4.21.

    Find the derivative of the function \(\dfrac{1.356}{1+20.5e^{-4.6t}}\)
    Solution.
    \(1.356\cdot -(1+20.5e^{-4.6t})^{-2}\cdot (20.5e^{-4.6t}\cdot -4.6)\)

    Example 3.4.22.

    Compute the derivative of \(j(x) = \ln (\ln (\ln (x^2-e^{3x})))\)
    Solution.
    \(j'(x) = \dfrac{1}{\ln (\ln (x^2-e^{3x}))}\cdot \dfrac{1}{\ln (x^2-e^{3x})}\cdot \dfrac{1}{x^2-e^{3x}}\cdot (2x-3e^{3x})\)

    Example 3.4.23.

    The number of children under 18 living in households headed by a grandparent can be modeled as
    \begin{equation*} p(t) = 2.111e^{0.04t}\quad \text{million children} \end{equation*}
    where \(t\) is the number of years since 1980
    1. Write the rate-of-change formula for \(p\text{.}\)
    2. How rapidly was the number of children living with their grandparents growing in 2010?
    Solution.
    1. \(\displaystyle p'(t) = 2.111\cdot 0.04e^{0.04t}\)
    2. \(p'(30) = 0.28\) million children per year

    Example 3.4.24.

    The tuition \(x\) years from now at OU is projected to be \(t(x) = 24072e^{0.056x}\) dollars.
    1. Write the rate-of-change formula for tuition.
    2. What is the rate of change in tuition four years from now?
    Solution.
    1. \(\displaystyle t'(x) = 24072\cdot 0.056e^{0.056x}\)
    2. \(t'(4) = 1686.48\) dollars per year
