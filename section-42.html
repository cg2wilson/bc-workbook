    Section 4.2 Relative Extreme Points

    Subsection 4.2.1 Definitions

    Definition 4.2.1. Relative Extrema.

    Let \(f(x)\) be a function defined on an input interval \([a,b]\text{.}\) Let \(a\lt c\lt b\text{.}\)
    We say that \(f\) has a relative maximum at \(c\) if the output \(f(c)\) is bigger than any other output in some interval around \(c\text{.}\) Likewise, \(f\) has a relative minimum at \(c\) if the output \(f(c)\) is smaller than any other output in some interval around \(c\text{.}\)
    Relative maxima/minima are also referred to as local maxima/minima or local extrema.

    Definition 4.2.2. Critical Point.

    A critical point of a continuous function \(f\) is a pair \((c,f(c))\) at which \(f\) is either not differentiable or has \(f'(c) = 0\text{.}\) The input value of \(c\) of a critical point is called the critical input.

    Example 4.2.3.

    Find the critical points of \(f(x) = 4x^3 + 8x^2 - 20x - 21\) on the interval \([-5,5]\text{.}\)
    Solution.
    From the calculator, we get \(x=-2.12\) and \(x = 0.79\) as the critical inputs; the critical points can be found by plugging these back into \(f(x)\text{.}\)

    Theorem 4.2.4. First Derivative Test.

    Suppose \(c\) is a critical input of a continuous function \(f\text{.}\)
    • If \(f'\) changes from positive to negative at \(c\text{,}\) then \(f\) has a relative maximum at \(c\text{.}\)
    • If \(f'\) changes from negative to positive at \(c\text{,}\) then \(f\) has a relative minimum at \(c\text{.}\)
    • If \(f'\) does not change its sign at \(c\text{,}\) then \(f \) has a horizontal tangent line at \(c\text{.}\)

    Theorem 4.2.5. Conditions Where Extreme Points Exist.

    For a function \(f\) with input \(x\text{,}\) a relative extremum can occur at \(x=c\) only if \(f(c)\) exists/is defined. Further,
    • A relative extremum exists where \(f'(c) = 0\) and the graph of \(f'(x)\) crosses the input axis at \(x=c\)
    • A relative extremum can exist where \(f(x)\) exists, but \(f'(x)\) does nto exist; further investigation will be needed.

    Theorem 4.2.6. Old Derivative Facts.

    Let \(f(x)\) be a function defined on an input interval \([a,b]\) and let \(a\lt c\lt b\)
    • If \(f'(x) \gt 0\text{,}\) then \(f(x)\) is increasing at \(x=c\text{.}\)
    • If \(f'(x) \lt 0\text{,}\) then \(f(x)\) is decreasing at \(x=c\text{.}\)
    • If \(f'(x) = 0\text{,}\) then the line tangent to \(f(x)\) at \(x=c\) is horizontal.

    Subsection 4.2.2 Methods of Finding Extrema

    Let \(f(x)\) be differenttiable on an open interval \((a,b)\text{.}\)
    Algebraic Method:
    1. Calculate the derivative \(f'(x)\)
    2. Set \(f'(x) = 0\) and solve for \(x\text{.}\) All solutions will be horizontal asymptotes; individual analysis (corresponding to 1st Derivative Test) will determine if \(f'(c)\) is a maximum, minimum, or neither.
    Calculator, Method 1
    1. Input \(f(x)\) into \(Y_1\)
    2. Plot \(f(x)\text{,}\) and do Zoom\(\rightarrow\)0:ZoomFit
    3. If you are finding a local maximum, press 2nd\(\to\)Trace\(\to\)4:maximum. If you are finding a local minimum, press 2nd\(\to\)Trace\(\to\)3:minimum.
      1. The calculator will prompt for a left bound. Input a number slightly less than where you expect the maximum/minimum to be.
      2. The calculator will then prompt for a right bound. Input a number slightly greater than where you expect the maximum/minimum to be.
      3. The calculator will then prompt for a guess. Input a guess, or press enter.
      4. The max/min will be displayed as a coordinate pair. If the pair is needed, use appropriate rounding guidelines.
      5. If you forget the value of the max/min, the calculator will store the \(x-\)coordinate in the variable \(X\text{.}\) In order to recall it, on the home screen, press \(X\) and the calculator will return the appropriate value.
    Calculator, Method 2
    1. Input \(f(x)\) into \(Y_1\)
    2. In \(Y_2\text{,}\) use the nDeriv command by pressing Math\(\to\)nDeriv(. The complete entry must look like
      							Y_2 = nDeriv(Y1(X),X,X)
      This will have the calculator graph the derivative as well as the original function
    3. The local extrema are given by wherever the derivative graph touches the \(x-\)axis. According to 1st Derivative Test, a local max occurs when the derivative crosses from positive to negative, and a local min occurs when the derivative crosses from negative to positive.

    Subsection 4.2.3 Examples

    Example 4.2.7.

    The percentage of people in the United States (aged 15 and above) who are sleeping at a given time of night can be modeled as
    \begin{equation*} s(t) = -2.63t^2 + 29.52t + 13.52\text{ percent, }t\text{ number of hours after 9pm} \end{equation*}
    1. Find the critical input values of \(s\) on the interval \(0\leq t\leq 11\text{,}\) and calculate the output value for any critical point.
    2. Graph \(s(t)\) and \(s'(t)\text{.}\) Label and interpret the critical inputs.
    Solution.
    1. The input is at \(t=5.61\text{,}\) and the critical point is \((5.61,96.36)\)
    2. The graphing can and should be done on the calculator. This is a good place to really emphasize the interval given, so that you set up what happens in Section 4.3.
      5.61 hours after 9pm, the percentage of people age 15 and above in the US who are sleeping is at a maximum, 96.36%

    Example 4.2.8.

    Sketch a graph such that
    • \(f\) has a relative maximum at \(x = 3\)
    • \(f\) has a relative minimum at \(x = -1\)
    • \(f'(x) \lt 0\) for \(x \lt -1\) and \(x \gt 3\)
    • \(f'(x) \gt 0\) for \(-1 \lt x \lt 3\)
    • \(\displaystyle f'(-1) = f'(3) = 0\)
    Solution.
    Answers vary, but the easiest solution is a downward quadratic whose vertex is at \(x=-1\text{.}\)

    Example 4.2.9.

    Consider the function \(f(x) = x^2 + 2.5x - 6\text{.}\)
    1. Write the derivative formula
    2. Locate and classify each critical point.
    Solution.
    1. \(\displaystyle f'(x) = 2x+2.5\)
    2. \((-1.25,-7.56)\) is a relative minimum

    Example 4.2.10.

    Consider the function \(f(x) = 0.3x^3 + 1.2x^2 - 6x + 4\text{.}\)
    1. Write the derivative formula
    2. Locate and classify each critical point.
    Solution.
    1. \(\displaystyle f'(x) = 0.9x^2+4.2x-6\)
    2. The critical point \((-4.24,28.15)\) is a relative max and \((1.57,-1.30)\) is a relative min.

    Example 4.2.11.

    Consider the function \(f(x) = 5e^{-x} + \ln x\) (for \(x \gt 0\)).
    1. Write the derivative formula
    2. Locate and classify each critical point.
    Solution.
    1. \(\displaystyle f'(x) = -5e^{-x} + \dfrac{1}{x}\)
    2. The critical point \((0.3,2.51)\) is a relative max and \((2.54,1.33)\) is a relative min

    Example 4.2.12.

    Consider the function \(f(x) = \dfrac{10}{1 + 2e^{-0.5x}}\text{.}\)
    1. Write the derivative formula
    2. Locate and classify each critical point.
    Solution.
    1. \(\displaystyle f'(x) = -10(1+2e^{-0.5x})^{-2}(-e^{-0.5x})\)
    2. None

    • \(f'(x) \gt 0\) for \(x \gt 2\)
    • \(f'(x) = 0\) for \(x = 2\)
