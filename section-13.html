    Section 1.3 Function Behavior and End Behavior Limits

    Subsection 1.3.1 Motivating Example

    Consider the function \(f(x) = \dfrac{1}{x-2}\text{,}\) graphed below:
    1. What happens to the output of \(f\) as the input increases without bound? Write your answer in limit notation.
      \(\ds \lim_{x\to\infty} f(x) = 0\)
    2. What happens to the output of \(f\) as the input decreases without bound? Write your answer in limit notation.
      \(\ds \lim_{x\to-\infty} f(x) = 0\)
    3. There is a vertical asymptote; the outputs increase or decrease without bound depending on which side of 2 that you're on.
    Ultimately, the idea is that the output of the function around \(x=2\) depends on which side you approach from; this necessitates the idea of a right and left-hand limit.

    Subsection 1.3.2 Left/Right Hand Limit

    Definition 1.3.1. Left/Right Hand Limit.

    Let \(f\) be a function defined on an interval containing some constant \(c\) (except possibly at \(c\) itself).
    • If \(f(x)\) approaches the value of \(L_1\) as \(x\) approaches \(c\) from the left, then the left-hand limit of \(f\) is \(L_1\text{,}\) and is written \(\ds \lim_{x\to c^-} f(x) = L_1\)
    • If \(f(x)\) approaches the value of \(L_2\) as \(x\) approaches \(c\) from the right, then the right-hand limit of \(f\) is \(L_2\text{,}\) and is written \(\ds \lim_{x\to c^+} f(x) = L_2\)

    Example 1.3.2.

    For \(f(x) = \dfrac{1}{x-2}\text{,}\) find \(\ds \lim_{x\to 2^-} f(x)\) and \(\ds \lim_{x\to 2^+} f(x)\text{.}\)
    Solution.
    \(\ds \lim_{x\to 2^-} f(x) = -\infty\) and \(\ds \lim_{x\to 2^+} f(x) = \infty\)

    Subsection 1.3.3 Continuity

    Definition 1.3.8.

    A function \(f(x)\text{,}\) defined on some input interval containing \(c\text{,}\) is said to be continuity at \(c\) if and only if the following conditions are satisfied:
    1. \(\ds \lim_{x\to c} f(x)\) exists
    2. \(f(c)\) exists
    3. \(\displaystyle \ds \lim_{x\to c} f(x) = f(c)\)
    A function is continuous on any open interval \((a,b)\) if it is continuous at every point inside the interval. If a function is not continuous at the input \(x = c\text{,}\) then we say that \(f\) is discontinuous at \(c\text{.}\)
    It's probably worthwhile to point out the explicit connection between the first condition and left/right-hand limits.

    Example 1.3.9.

    Identify any points of discontinuity in the function \(f(x) = \dfrac{1}{x-2}\text{.}\) Explain why the function is discontinuous at those points.
    Solution.
    \(f(x)\) is discontinuous at \(x=2\) because it fails the first and second conditions of continuity.

    Example 1.3.10.

    Identify any points of discontinuity in the function \(g(x)\) in Example 1.3.4. Explain why the function is discontinuous at those points.
    Solution.
    There's a discontinuity at \(x=3\) because \(\ds \lim_{x\to 3} g(x)\) doesn't exist.

    Subsection 1.3.4 Properties of Limits

    Let \(f(x)\) and \(g(x)\) be continuous functions over some input interval containing \(c\text{,}\) and \(k\) be some arbitrary constant. Then, we have the following properties of limits:
    1. Constant Rule: \(\ds \lim_{x\to c} k = k\)
    2. Sum Rule: \(\ds \lim_{x\to c}\left[f(x) + g(x)\right] \)
    3. Constant Multiple Rule: \(\ds \lim_{x\to c} \left[k\cdot f(x)\right] = k\lim_{x\to c} f(x) \)
    4. Replacement Rule: If \(f(c)\) is defined at \(c\text{,}\) then \(\ds \lim_{x\to c}f(x) = f(c)\text{.}\)
    5. Product Rule: \(\ds\lim_{x\to c} \left[f(x)\cdot g(x)\right] = \left[\lim_{x\to c} f(x)\right]\cdot\left[\lim_{x\to c}g(x)\right]\)
    6. Quotient Rule: \(\ds \lim_{x\to c} \left[\dfrac{f(x)}{g(x)}\right] = \dfrac{\ds\lim_{x\to c} f(x)}{\ds\lim_{x\to c} g(x)}\) (as long as \(\ds\lim_{x\to c} g(x)\neq 0\))
    7. If \(f(x)\) can be factored as \(f(x) = h(x)\cdot k(x)\text{,}\) and \(g(x)\) can also be factored as \(g(x) = j(x)\cdot k(x)\text{,}\) then
      \begin{equation*} \lim_{x\to c} \frac{f(x)}{g(x)} = \lim_{x\to c}\frac{h(x)\cdot k(x)}{j(x) \cdot k(x)} = \lim_{x\to c} \frac{h(x)}{j(x)} \end{equation*}
      i.e. common factors may be canceled across fractions under the limit.
    I don't know how I missed the "Difference Rule", but it should be lumped in with the Sum Rule.
    I think the Replacement Rule should read "If \(f(c)\) is continuous" rather than "defined".

    Let \(f(x) = \begin{cases} x^2 & x \lt -1 \\ 1 & x\geq -1\end{cases}\text{.}\) Algebraically determine the following limits and answer the questions:
